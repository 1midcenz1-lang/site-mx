# مهاجرت دقیق Flask به PHP (نسخه عملیاتی)

درخواست شما: **ظاهر و رفتار سایت دقیقاً مثل نسخه Python/Flask باشد ولی روی هاست PHP اجرا شود.**

این commit لایه PHP را به شکل واقعی‌تر جلو برده:
- routing برای مسیرهای اصلی اضافه شد
- layout و کلاس‌های CSS نسخه فعلی حفظ شد
- فایل‌های JS فعلی پروژه لود می‌شوند

## وضعیت فعلی

- ظاهر کلی صفحات اصلی روی PHP رندر می‌شود.
- برای یکسان شدن کامل رفتار، باید endpointهای Flask به PHP تبدیل شوند (الان JSها هنوز endpointهای Flask را صدا می‌زنند).

## مسیرهایی که آماده شدند

- `/`
- `/login`
- `/buy`, `/buy/irani`, `/buy/khareji`
- `/my-videos`
- `/messages`
- `/admin`, `/admin/login`

## برای ۱۰۰٪ یکسان شدن عملکرد باید این APIها در PHP پیاده‌سازی شوند

- خرید/آپلود فیش
- مدیریت دسترسی دسته‌ها
- ادمین و تایید/رد خرید
- پیام‌ها، ریپورت‌ها، لایک‌ها، آمار عمومی
