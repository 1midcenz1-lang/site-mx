# راه‌اندازی نسخه PHP برای هاست اشتراکی

این پروژه الان با **Flask (Python)** نوشته شده و روی هاست فقط-PHP مستقیم اجرا نمی‌شود.
برای اینکه سایت روی هاست PHP بالا بیاید، این ریپو یک لایه‌ی سازگار PHP دریافت کرده که:

1. فایل‌های `static/` را مستقیم سرو می‌کند.
2. مسیرها را با `index.php` هندل می‌کند.
3. صفحه‌ها را از `templates/` به خروجی HTML ساده تبدیل‌شده ارائه می‌دهد.

## محدودیت مهم

این تبدیل، **هسته‌ی بک‌اند Flask** (مدیریت خرید، ادمین، آپلود، APIها) را ۱۰۰٪ جایگزین نمی‌کند.
اگر هاست شما فقط PHP است، باید بک‌اند کامل به PHP (مثلاً Laravel یا PHP خالص) مهاجرت شود.

## Quick Deploy

- کل پروژه را آپلود کنید.
- `index.php` و `.htaccess` باید در ریشه‌ی سایت باشند.
- روی Apache، `mod_rewrite` فعال باشد.

## مسیرهای آماده در لایه PHP

- `/` → صفحه اصلی ساده
- `/login` → صفحه ورود ساده
- `/buy` → صفحه خرید ساده

## قدم بعدی برای مهاجرت کامل

- بازنویسی endpointهای Flask در PHP:
  - احراز هویت ادمین
  - آپلود فیش
  - مدیریت دسته‌ها و دسترسی‌ها
  - پیام‌ها و ریپورت‌ها
  - APIهای نظرسنجی/لایک/آمار
